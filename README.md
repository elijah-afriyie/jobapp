# jobapp

# JOB

### FIELDS
* Id
* Job title
* Description
* Minimum salary
* Maximum salary
* Location

### END POINTS
* GET /job
* GET /job/{id}
* POST /job
* PUT /job/{id}
* DELETE /job/{id}

### END POINTS

### END POINTS

* DELETE /companies/{companyId}/reviews/{reviewsId}
